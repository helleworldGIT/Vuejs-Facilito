# vueyecto

## Primero, debes instalar el proyecto on
```
npm install
```

### Y luego, para poder trabajar en el proyecto, utilizar el comando:
```
npm run serve
```

### Lo último es hacer la build para producción UNA VEZ EL PROYECTO FUNCIONE
```
npm run build
```
